What is Dubnium?

Dubnium is a package available on NPM. Dubnium allows you to create local databases perfect for storing a few values for an app's configuration or thousands of User's data on a webserver. However you need to store your data, Dubnium can help you out.

Why use Dubnium?

Read Dubnium's key features page

How do I use Dubnium?

First, install it from NPM, [sudo] npm i dubnium -g. Then require or import (Commonjs & ESM are supported) it in your project! If you need extended help, read our docs.
